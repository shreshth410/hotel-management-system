# Hotel Management System
A desktop **Hotel Management System** built using **Java and JavaFX**.  
The application provides a graphical interface to manage hotel operations such as rooms, customers, bookings, and billing.

## Features

- Room management (add, view, update availability)
- Customer management
- Room booking and checkout
- Automatic billing calculation
- Persistent data storage
- JavaFX GUI designed with Scene Builder
- Clean, modular project structure using Maven

## Technologies Used

- Java
- JavaFX
- Maven
- FXML / Scene Builder
